Навчальний проект - Агрегатор погодних сайтів.

Мета.
На практиці навчитися створювати додатки на мові програмування Python, які 
мають практичну цінність.

Ідея.
Створити програму, яка буде отримувати дані з декількох погодних сайтів та 
відображати їх у зручному вигляді в додатку.

Реалізація.
Для отримання даних з погодних сайтів буде використаний метод Web scraping. 
Програма:
    - буде консольною;
    - повинна мати простий інтерфейс користування;
    - модульну систему, для можливості створення додаткового функціоналу без 
    зміни основного функціоналу;
    - повинна бути реалізована так, щоб була можливість подальшого розвитку і 
    мати можливість створення повноцінного веб сервісу.
